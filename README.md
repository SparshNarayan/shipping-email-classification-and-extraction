# shipping-email-classification-and-extraction
🚢 AI-Powered Shipping Email Segregation and Data Extraction System
📌 Overview

This project is an AI-powered maritime email processing system that automatically classifies shipping emails into Tonnage, Cargo VC (Voyage Charter), and Cargo TC (Time Charter) categories. The system extracts key shipping information and converts unstructured emails into structured datasets for easier analysis and decision-making.

✨ Features
Automated Shipping Email Classification
TF-IDF based Text Vectorization
Multinomial Naive Bayes Classification
Regex-based Information Extraction
Vessel and Cargo Database Management
Vessel–Cargo Matching Module
CSV Export Functionality
Interactive Web Dashboard
Offline Prediction (No External API Required)
